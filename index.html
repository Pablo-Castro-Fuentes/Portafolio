Sobre mí

Soy Pablo Castro, un técnico en informática apasionado por el desarrollo web y la inteligencia artificial. Mi interés por la tecnología comenzó con los videojuegos y la curiosidad de entender cómo funcionan, lo que me llevó a explorar lenguajes como Python y Java para crear mis propias soluciones. Me fascina cómo la IA puede transformar ideas en herramientas prácticas, y disfruto experimentando con frameworks como Angular o Node.JS para construir interfaces dinámicas.

Aprendo mejor desarmando problemas complejos en partes pequeñas, ya sea resolviendo ejercicios de matemáticas o depurando código en Visual Studio Code. Mis fortalezas están en mi capacidad para adaptarme a nuevas tecnologías, mi dominio del inglés para investigar documentación y mi enfoque lógico para gestionar bases de datos como MySQL u Oracle SQL. Cuando no estoy programando, me conecto a la Realidad Virtual, para aumentar mis habilidades sociales.
